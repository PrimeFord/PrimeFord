## 🚀 Featured Project — FarmNomad

> **A modular AgriTech platform connecting farmers, buyers, drivers, food banks, and administrators in a single coordinated system.**

FarmNomad addresses a real gap in UK rural food systems — the absence of an integrated, role-aware platform that combines agricultural e-commerce, automated surplus detection, and donation-to-delivery logistics.

|Layer   |Technology                                |
|--------|------------------------------------------|
|Backend |Java, Spring Boot, Spring Security, JWT   |
|Frontend|React, Next.js, Redux Toolkit, TailwindCSS|
|Database|PostgreSQL                                |
|Payments|Stripe Checkout & Webhooks                |
|DevOps  |Docker, CI/CD, GitHub Actions             |
|Testing |JUnit 5, Mockito, Testcontainers          |

🔗 **Live:** [farmnomad.vercel.app](https://farmnomad.vercel.app/)
